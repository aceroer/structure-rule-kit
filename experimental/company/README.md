# Agent Company Experimental Layer

This folder tracks the transition from Agent GitHub Worknet toward Agent
Company.

Agent GitHub Worknet remains the GitHub-native substrate:

```text
issues -> tasks -> branches -> sessions -> verification -> PR-ready records
```

Agent Company is the upper layer:

```text
offices -> authority -> methods -> reports -> meetings -> gates -> metrics
```

The purpose of this experimental layer is to keep external influences,
governance traces, and office-method prototypes visible before they become
stable CLI commands or templates.

## Contents

```text
experimental/company/
├── README.md
└── external/
    └── gstack.md
```

## Rule

External projects may inspire office methods, but Agent Company must preserve
its own governance model:

- explicit authority
- role-owned artifacts
- capability-token gates
- publication boundaries
- auditable traces
- human takeover points

Skill packs are methods. Agent Company is the organization runtime.
